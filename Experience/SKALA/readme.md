# SKALA

SKALA 교육 과정에서 수행한 실습 및 프로젝트 자료 저장소입니다.

## 디렉토리 구조

### [cloud/](cloud/)

클라우드 및 컨테이너 관련 실습

- Docker 배포 실습

### [data-science/](data-science/)

데이터 분석 및 데이터 사이언스 프로젝트

- 리뷰 데이터 분석 보고서

### [dbms/](dbms/)

데이터베이스 관리 시스템 관련 실습 및 프로젝트

- PostgreSQL 락(Lock) 및 모니터링 (`pg_locks`, `pg_stat_activity`)
- 권한 및 ROLE 관리
- 잠금 대기 및 Deadlock 상황 감지
- 벡터 인덱싱 성능 비교
- CSV 데이터 PostgreSQL 적재 (API, UI)
- Tableau/Metabase 연동
- Github Issues 벡터 검색 시스템
- 백업 및 복구(Backup & Restore)
- MariaDB 설정 가이드

### [etc/](etc/)

기타 문서 및 참고 자료

- 보고서 작성 요령

### [fastapi/](fastapi/)

FastAPI 프레임워크를 활용한 백엔드 개발

- 주식 거래 시스템

### [html-css/](html-css/)

웹 프론트엔드 기초

### [javascript/](javascript/)

JavaScript 및 Java 프로그래밍

### [linux/](linux/)

리눅스 기초 및 시스템 관리

### [llm/](llm/)

대규모 언어 모델(LLM) 관련 학습

### [ml-dl/](ml-dl/)

머신러닝 및 딥러닝 실습

- Transformer 보고서
- Machine Learning (회귀, 분류)
- CNN (Convolutional Neural Networks)
- CIFAR-10 분류 및 리팩토링
- LSTM (Long Short-Term Memory)
- Autoencoder
- ResNet
- BERT
- 하이퍼파라미터 최적화 (Hyperparameter Optimization)
- Explainable AI (XAI)

### [python/](python/)

Python 프로그래밍 실습

- Python 기초 실습
- 제너레이터(Generator)
- 로그 출력

- 데코레이터(Decorator)

RAG (Retrieval-Augmented Generation) 시스템 구현

SQL 및 데이터베이스 활용

## 주요 기술 스택

- **언어**: Python, JavaScript, Java, SQL
- **데이터베이스**: PostgreSQL, MariaDB
- **웹 프레임워크**: FastAPI, Streamlit
- **머신러닝/딥러닝**: TensorFlow, PyTorch, Scikit-learn
- **벡터 DB**: FAISS, Chroma, pgVector
- **시각화**: Tableau, Metabase
- **컨테이너**: Docker
- **기타**: RAG, LLM, Transformer, BERT

## 학습 내용

이 저장소는 다음과 같은 주제를 다룹니다:

1. **백엔드 개발**: FastAPI를 활용한 REST API 개발
2. **데이터베이스**: PostgreSQL/MariaDB 관리, 최적화, 벡터 검색
3. **머신러닝/딥러닝**: 회귀, 분류, CNN, RNN, Transformer 등 다양한 모델 구현
4. **AI 응용**: RAG 시스템, LLM 활용, 벡터 검색 시스템
5. **웹 개발**: HTML/CSS/JavaScript 기초부터 실전 프로젝트까지
6. **시스템 관리**: Linux 기초, Docker 컨테이너화
7. **데이터 분석**: 리뷰 데이터 분석, 시각화
