# 쇼핑몰 웹페이지 실습

실습 내용
- 당신은 온라인 쇼핑몰의 개발자로, 고객 주문을 처리하는 프로그램을 작성하고 있습니다. 주문 처리 과정에서는 여러 조건을 고려해야 합니다. 예를 들어, 상품의 재고 여부, 고객의 회원 등급, 주문 금액, 배송 옵션 등을 확인하여 적절한 메시지와 할인율을 적용해야 합니다.
- 하기 세부 조건에 맞도록 Javascript 함수를 구현하시고 최종 출력을 console.log 에 출력되도록 해 주세요.

세부 조건
- 상품 재고 확인
- 재고가 1개 이상일 때: 주문을 진행한다.
- 재고가 0개일 때: 품절 메시지를 출력한다.

회원 등급에 따른 할인
- “VIP” 회원: 10% 할인.
- “Gold” 회원: 5% 할인.
- 그 외 회원: 할인 없음.

주문 금액에 따른 추가 할인
- 기본 할인 후 주문 금액이 100,000원 이상이면 5,000원 추가 할인.
- 기본 할인 후 주문 금액이 200,000원 이상이면 15,000원 추가 할인.

배송 옵션
- “fast”(빠른 배송) 옵션을 선택하면 배송비 3,000원 추가.
- “standard”(일반 배송) 또는 배송 옵션을 선택하지 않으면 배송비 0원.

최종 출력 메시지
- 주문이 성공적으로 처리될 경우: “주문이 정상적으로 완료되었습니다.”라는 메시지와 함께 최종 결제 금액을 출력한다.
- 재고가 없을 경우: “죄송합니다. 해당 상품은 품절입니다.”라는 메시지를 출력한다.

추가 옵션
- 위 코드를 반영한 HTML 페이지 내 상품재고, 회원등급, 최초주문금액, 배송옵션을 입력 받아 최종 출력 메시지를 alert 으로 띄워지게 해주세요.
  - 상품재고 및 최초주문금액을 반드시 숫자가 입력되도록 해야 함.
  - 회원등급과 배송옵션은 Drop-down List 형태로 반영해야 함.
  - 버튼 클릭하면 해당 입력 값의 유효성 체크 후 함수 호출되도록 해야 함.
  - 함수 호출 및 함수 내 로직 수행 후 최종 출력 메시지가 alert() 을 활용하여 보여져야 함.
  - CSS도 일부 적용 되도록 해야 함. (선택자, 속성, 값)
