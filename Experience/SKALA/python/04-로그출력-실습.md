# python - 로그 출력 실습

1단계: .env 작성
- env 파일을 만들고 다음 키-값을 저장
    - LOG_LEVEL=???
    - APP_NAME=???
- LOG_LEVEL에는 DEBUG 또는 INFO,
- APP_NAME에는 자유로운 앱 이름 입력

2단계: 환경 변수 로드
- python-dotenv 모듈 설치
    - pip install python-dotenv
- .env 파일을 읽어오는 코드 작성 (힌트: load_dotenv() 사용)
- os.getenv("LOG_LEVEL") 로 변수 읽기

3단계: logging 설정
- 로그 파일명: app.log
- 로그 포맷: 시간 [레벨] 메시지
- 로그 레벨: .env에서 가져온 값 사용 (getattr(logging, log_level))
- 콘솔 + 파일 동시에 출력되도록 핸들러 구성

4단계: 로그 출력 테스트
- INFO 레벨 메시지: "앱 실행 시작"
- DEBUG 레벨 메시지: "환경 변수 로딩 완료"
- ERROR 레벨 메시지: ZeroDivisionError 예외 발생 시 출력

5단계: 실행 후 확인
- 콘솔 화면 출력 확인
- app.log 파일 내용 확인

콘솔 화면의 결과와 env 파일 내용, app.log에 기록된 내용, 그리고 개별로 작업하셨던 폴더의 구조에 대해 화면 Capture 하셔서 댓글로 올려주세요.
